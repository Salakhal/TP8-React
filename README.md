# 🌸 TP8 - Consommer une API avec Fetch et Axios dans React



## 📋 Description

Ce projet est un TP d'apprentissage qui démontre comment consommer des API REST dans une application React en utilisant deux méthodes différentes :

- **Fetch API** : La méthode native du navigateur
- **Axios** : Une librairie externe plus puissante

L'application récupère des données depuis l'API publique [JSONPlaceholder](https://jsonplaceholder.typicode.com/) et les affiche dans une interface élégante avec un thème rose.

## 🎯 Objectifs du TP

- ✅ Comprendre la différence entre `fetch()` et `axios`
- ✅ Charger des données depuis une API externe
- ✅ Afficher des listes dynamiques dans React
- ✅ Gérer les états de chargement (loading)
- ✅ Gérer les erreurs de manière élégante
- ✅ Structurer proprement son code React
- ✅ Créer une interface utilisateur responsive et moderne

## ✨ Fonctionnalités

### Fetch API
- 📰 Affichage des articles (posts)
- 🔍 Recherche en temps réel
- 🔄 Rechargement des données
- ⏳ Indicateur de chargement
- ❌ Gestion des erreurs

### Axios
- 👥 Affichage des utilisateurs
- 🖱️ Clic pour afficher les détails
- 🔄 Rechargement des données
- ⏳ Indicateur de chargement
- ❌ Gestion des erreurs

### Interface
- 🎨 Design moderne avec thème rose
- 📱 Responsive (mobile, tablette, desktop)
- 🧭 Navigation par onglets
- ✨ Animations fluides
